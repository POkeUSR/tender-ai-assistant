1. Язык и среда

Python

Причина: вся экосистема AI-документов вокруг него.

Установить:

Python 3.10+

pip

virtualenv

2. LLM (мозг системы)

Используйте:

OpenAI GPT-4o

Он стабильно работает с:

юридическими текстами

длинными документами

структурированными ответами

Позже можно добавить fallback:

Claude 3.5 Sonnet

3. Фреймворк для RAG и агентов

Лучший старт:

LangChain

Он даст:

RAG

агенты

tool calling

memory

4. Векторная база

Для MVP не нужен сервер.

Используйте:

FAISS

Плюсы:

работает локально

быстрый

простой

Позже можно перейти на:

Pinecone

5. Embeddings

Используйте embeddings от OpenAI.

Модель:

text-embedding-3-large

Она хорошо работает с длинными документами.

6. Загрузка PDF

Нужен стабильный парсер.

Используйте:

pdfplumber

или

PyMuPDF

Для тендеров PyMuPDF часто лучше.

7. Chunking документа

Лучший инструмент:

RecursiveCharacterTextSplitter

из:

LangChain

Настройки:

chunk_size = 1200
chunk_overlap = 200
8. Интерфейс для демо

Streamlit

Он позволяет за несколько часов сделать UI:

upload PDF
анализ
чат
